# 高性价比机场入口说明页索引

本仓库用于托管一个静态索引页面，用于汇总各品牌的「地址说明与入口指引」页面，便于用户在地址变更或无法访问时进行备用访问与参考。

该页面仅用于信息展示与访问指引，不提供注册、登录、购买或任何形式的服务功能。

## 索引地址

https://jichangbaike.github.io/

## 内容说明

当前索引页面包含以下类型的内容：

- 各品牌地址说明页的链接
- 用于帮助用户找到最新可用访问入口的参考信息

不包含：

- 服务注册或登录功能  
- 用户数据收集、表单或追踪脚本  
- 任何形式的支付、推广或营销功能  

## 使用目的

该页面的设计目标是：

- 提供一个稳定的备用入口索引页
- 作为地址说明页的聚合与跳转目录
- 帮助用户在地址变更时仍可找到对应页面
